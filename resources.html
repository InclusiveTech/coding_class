Lightbulb

Create lightswitch by using the HTML DOM to assign an "onclick" event to a button element. Copy this code into your editor to get started!

<html>
<body>
	<h1>JavaScript Power</h1>

	<p>Power this lightbulb by using the HTML DOM to assign an "onclick" event to a button element.</p>

	<button>Turn on the light</button>

	<img id="lightbulb" src="light_off.png" style="width:100px">

	<button>Turn off the light</button>
</body>
</html>

Here's an example of a request you could make to the Open Weather Map API. You'll want to replace <API_key> with a real API key. This request will get us data about the weather in London, with the units in metric (i.e. degrees Celcius):

https://api.openweathermap.org/data/2.5/weather?q=London&units=metric&APPID=<API_key>
